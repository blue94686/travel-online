# UI Template

The interface uses a restrained scenic travel style:

- Deep ink-green hero and admin sidebar.
- Real landscape photography with fixed aspect ratios.
- White content areas, light borders, restrained shadows.
- Lake-green primary buttons and orange scoring or reason tags.
- Desktop content is centered with a 1440px maximum width.
- Home and admin sections use at most two major modules per row.
- Mobile layouts collapse to one column.
